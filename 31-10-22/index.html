Javascript e gli Array
Esercizio 5


ESERCIZIO

Sulla base della lezione del giorno, dato un array di oggetti definito (riportato nel file readme.md), processarlo al fine di stampare solo il valore delle seguenti chiavi (tramite `console.log`):

  • title
  • price
  • count

### Attenzione, l'esercizio si ritiene completo se:

  • si utilizzano solo arrow functions
  • si processa con almeno due HOF (Higher-Order Functions) tra `forEach`, `map` e `filter`
  • scrivere alla fine dell'esercizio, in un commento, tutte le callbacks all'interno del vostro esercizio

    • Il risultato finale dovrà essere qualcosa del genere:
    ```
    -> Fjallraven - Foldsack No. 1 Backpack, Fits 15 Laptops
    -> 109.95
    -> 120
    -> -------------------
    -> Mens Casual Premium Slim Fit T-Shirts
    -> 22.3
    -> 259
    -> -------------------
    -> ...
    -> ...
    -> ..
    ```
